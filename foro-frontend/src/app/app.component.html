Sobre este foro
Este foro fue desarrollado como proyecto fullstack utilizando:
  • Frontend: Angular 17 + Bootstrap 5
  • Backend: Java Spring Boot 3 (API REST)
  • Base de datos: H2 (modo archivo, solo para demo)
  • Deploy Frontend: Vercel
  • Deploy Backend: Render.com
Nota importante: La primera vez que accedes o tras un tiempo de inactividad, el backend puede demorar hasta 30 segundos en responder porque Render "duerme" servicios gratuitos cuando no hay actividad. Los siguientes accesos serán mucho más rápidos.
